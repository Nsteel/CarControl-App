About Car Control

This app is an adaptation of the open source project ROS Control
and it was developed for the Projektseminar Echtzeitsysteme (PSES) at the TU-Darmstadt.
Our goal was to create an Android application to interface with the Robot Operating System (ROS) and the model car for direct drive, diagnostic, and navigation information. The application processes data from various sensors on the car and displays the information in an understandable fashion. It has been optimized for tablets whose increased screen size can accomodate the variety of displays for visualizing data from the car's sensors.
